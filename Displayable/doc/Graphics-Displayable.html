class Displayable t where

The type class of things that can be displayed.

For the click function, the first point is the location of the click, and the second point is the upper-left corner of the thing being clicked.

The measure function gives a fixed dimension for all values of a type, appropriate for table formatting. The flowMeasure function gives a specific measurement for each value, appropriate for a flow layout.
